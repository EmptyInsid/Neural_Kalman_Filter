# Neural Kalman Filter

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PyTorch](https://img.shields.io/badge/PyTorch-2.0+-red.svg)](https://pytorch.org/)

**Адаптивный нейросетевой фильтр Калмана для динамической оценки состояния в условиях нестационарного шума.**

> Курсовая работа. СПбПУ, Институт компьютерных наук и кибербезопасности, 2026.

Проект исследует гибридный подход: классический фильтр Калмана, усиленный нейронной сетью, которая динамически оценивает дисперсию шума измерений. Особый акцент сделан на сравнении оптимизаторов первого (Adam) и второго порядка (Экспоненциальная релаксация) в задачах со сложным ландшафтом функции потерь.

---

## Ключевые особенности

*   **Гибридная архитектура**: Нейросеть (MLP) заменяет не адаптацию шума, а полную аппроксимацию фильтра. Это сохраняет интерпретируемость классического алгоритма.
*   **Адаптация к шуму**: Сеть на лету пересчитывает `R` (ковариацию шума измерений) на основе окна из 5 последних инноваций (residual).
*   **Продвинутая оптимизация**: Реализован метод **Экспоненциальной релаксации (ER)** — гибрид метода Ньютона и градиентного спуска с адаптивным демпфированием и линейным поиском Армихо.
*   **Сравнительный анализ**: Встроенный бенчмарк для классического Калмана, нейросети на Adam и нейросети на ER.
*   **Интерактивный GUI**: Визуализация траекторий, логов сходимости, настройка архитектуры сети и функций потерь (MSE, MAE, Huber).

## Быстрый старт

### Требования
- Python ≥ 3.11.2
- Рекомендуется виртуальное окружение

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/EmptyInsid/Neural_Kalman_Filter.git
cd Neural_Kalman_Filter

# Создание и активация виртуального окружения
python3 -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt
```

### Запуск

```bash
# Классический фильтр Калмана (без нейросети)
python -m app.classic_calman_app.gui.gui

# Нейросетевой фильтр Калмана (основное приложение)
python -m app.neural_calman_app.gui.gui
```

## Основные результаты

1. **Адаптивность к нестационарному шуму**:
Нейросетевой фильтр динамически подстраивает коэффициент усиления Калмана при ступенчатом изменении дисперсии шума датчика (σ_η), в то время как классический Калман с фиксированным R теряет оптимальность.

2. **Превосходство метода ER в "овражных" задачах**:
При моделировании манёвренного объекта (сложный нелинейный ландшафт) оптимизатор Экспоненциальной релаксации достигает на порядок меньшей ошибки (RMSE), чем Adam, за счёт использования кривизны Гессиана и линейного поиска.

3. **Требования к гладкости активаций**:
Метод ER критически зависит от дважды дифференцируемых активаций. Использование ReLU приводит к деградации (нулевые вторые производные). Наилучшие результаты показали Mish и GELU.

4. **Робастность к выбросам**:
Применение Huber Loss в паре с ER даёт лучшую устойчивость к аномалиям датчика, чем стандартная MSE.
